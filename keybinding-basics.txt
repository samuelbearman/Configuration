# How I do keybinds

- (VsCode) Editor View:
	- Ctrl+Shift+H: Previous Pane
	- Ctrl+Shift+L: Next Pane
	- Ctrl+Shift+A: New Terminal Session
	- Ctrl+Shift+B: Previous Terminal Session
	- Ctrl+Shift+J: Toggle to Terminal
	
- (VsCode) File Explorer View:
	- Ctrl+E: Toggle to editor
	- n: Create new file
	- Ctrl+n: Create new directory

- Windows Terminal:
	- Ctrl+Shift+H: Previous Pane
	- Ctrl+Shift+L: Next Pane
	- Ctrl+Shift+;: Split Vertical
	- Ctrl+Shift+': Split Horizontal
	- Ctrl+Shift+Up/Down/Left/Right: Move from split to split
	- Ctrl+Shift+N: New Terminal
	- Ctrl+Shift+M: Open Custom Terminal
	
- Vim:
	- Leader: Space
	- Comment toggle: Leader+C
	- Save: Leader+W
	- Exit: Leader+X
	- Quite: Leader+q
	- Highlight+>: Shift right
	- Highlight+<: Shift left
	- Highlight+K: Move lines up
	- Highlight+J: Move lines down
